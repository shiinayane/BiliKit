# 📺 BiliKit Feed — B站首页换成手机App个性化推荐流

> **B 站网页版首页推的东西不合口味？** 把它换成**手机 App 那套更懂你的推荐流**——还能点开就在**底部抽屉**里看、封面**悬停秒预览**，滚多久都**不卡**。

哔哩哔哩（bilibili / B站）首页增强油猴脚本，对 **macOS Safari** 友好，也支持 Chrome / Edge / Firefox。就地接管首页视频流，**登录后个性化、不重复**。

## ✨ 亮点

- 🎯 **App 个性化推荐** —— 用 B 站**手机 App 的推荐接口**替换网页版首页（网页版是 PC 推荐、往往不如 App 懂你）。扫码登录即获个性化、不重复的**信息流 / feed**。
- ⚡ **零框架 · 极致轻量** —— 纯原生 TypeScript + DOM 手写，**无 React / Vue 等任何运行时框架**，整个脚本 gzip 仅约 **22KB**（含自写的虚拟化与 MSE 预览）。性能是本脚本的第一卖点。
- 🚀 **窗口化虚拟化 · 滚动不卡** —— 只渲染可视区域的卡片、上下占位撑高，**DOM 数量恒定、长时间刷不涨内存**（对照重型 React 首页动辄上 G 内存）。锚点补偿滚动位置，上下翻**不抖动**。
- 🎬 **底部抽屉播放 · 不跳转** —— 点卡片从**底部滑出抽屉**、同源内嵌播放**完整视频页**（弹幕/评论/高清全在），首页不丢、不开一堆新标签。**顶部继续下拉即关**，或 Esc / 点缝。抽屉内**自动隐藏站内顶栏 + 去广告**、加载有封面遮罩、悬停预连接提速、主题跟随。
- 👀 **封面悬停「真视频」秒开预览** —— 鼠标停在封面上，用 **MSE** 只取头几秒分段**边播边补**，起播接近手机 App 原生（热门冷门都能播），静音循环、右下角显示「当前 / 总时长」。面板可切回**雪碧图缩略帧**或关闭。
- 🧱 **卡片信息齐全** —— 封面骨架微光加载、播放/弹幕/时长、UP 头像与名字（点进空间）、推荐理由徽章。
- 🔧 **打开方式可选** —— 底部抽屉 / 新标签页 / 当前页，面板里切换。右下角还有刷新内容 / 返回顶部。

## 需要搭配 BiliKit Core

本脚本负责首页推荐流与抽屉；**登录（获取个性化推荐）和设置面板由 [BiliKit Core] 提供**，请一并安装。Core 还带 CDN 优选（海外加速）、主题同步、评论性别/属地、防息屏。

## 安装与使用

1. Safari 装 [Userscripts](https://apps.apple.com/app/userscripts/id1463298887)；其它浏览器用 Tampermonkey / Violentmonkey。
2. 先装 **BiliKit Core**，再装本 **BiliKit Feed** → 刷新 B 站首页。
3. 打开 Core 面板「App 推荐 Feed」扫码登录 → 首页即变成个性化 App 推荐。
