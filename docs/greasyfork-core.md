# 🧩 BiliKit Core — B站体验增强核心（Safari 友好 / 一装到位）

> **海外看 B 站卡成幻灯片？没登录看不了评论 / 高清？看片总息屏？主题不跟系统？** 一个脚本全搞定。

哔哩哔哩（bilibili / B站）增强油猴脚本，专为 **macOS Safari** 打磨，同样支持 Chrome / Edge / Firefox。**无需浏览器扩展、零外部依赖**，装上即用，左下角一个齿轮打开**统一设置面板**，逐项开关。

## ✨ 六大功能

- 🚀 **CDN 优选 · 海外加速** —— 把视频取流重定向到你指定的**大陆镜像节点**，绕开海外常被分到的**慢 CDN / 回源失败**节点（HTTP 514）。专治**海外党卡顿、缓冲转圈、冷门视频加载不出**。改写 playurl 并重建备份镜像，杜绝主备线路间「URL 反复横跳」。
- 🛡️ **埋点 / 广告拦截 · 省流降耗** —— 拦掉 B 站的**行为遥测上报**与**广告投放**请求（`data.bilibili.com`、`cm.bilibili.com` 等），命中即短路、不发起。实测首页空闲时约 **65% 的请求是埋点 / 广告**，拦掉**零功能损失、页面无感**——省流量、降网络与主线程开销。广告拦截可单独关，支持自定义黑名单。
- 🌗 **主题同步 · 深色跟随系统** —— B 站全站（首页 + 播放页）**跟随系统深浅色**，**无刷新实时切换**、多标签页同步。告别手动切**夜间模式 / 暗色主题**。
- 📍 **评论属地** —— 在每条评论、回复的时间旁显示 **IP 属地 / 地区**。轻量实现，**替代那些会把视频页拖到几个 G 内存、让 Safari 卡死的第三方「开盒」脚本**。
- ☀️ **防息屏 · 防睡眠** —— 看视频时申请**屏幕常亮**（screen wake lock），暂停自动释放。再也不用中途去动一下鼠标防止息屏 / 睡眠。
- 🔓 **免登录看评论 / 动态 / 1080p**（默认关） —— 不登录也能看视频与动态下方**评论**、看他人**动态**、看 **1080P 高清**。装它即可替代 beefreely 等免登录脚本，**根除多个脚本抢改请求导致的时好时坏**。纯只读（发评论 / 点赞等仍需登录）；1080p 为官方试看清晰度。

## 为什么选它

- **Safari 友好**：页面世界注入（`@grant none`），才能真正拦到播放器请求、读到评论组件数据——很多同类脚本在 Safari Userscripts 下因权限被丢进隔离世界而**失效**。
- **零依赖、不重建页面**：不引第三方库、不碰 Vue 生命周期，轻量稳定。
- **统一设置面板**：所有功能集中开关、可配置（CDN 节点自定义、主题模式、属地前缀…）。

## 搭配 BiliKit Feed 更香

想把首页换成**手机 App 那种个性化推荐流**？再装一个 **BiliKit Feed**（点卡片底部抽屉播放、封面悬停预览、滚动流畅不卡）。Feed 的登录与设置就由本 Core 提供。

## 安装与使用

1. Safari 装 [Userscripts](https://apps.apple.com/app/userscripts/id1463298887)；其它浏览器用 Tampermonkey / Violentmonkey。
2. 安装本脚本 → 刷新 B 站 → 点**左下角齿轮**打开设置面板。
3. 海外卡顿：进面板「CDN 优选」选个大陆镜像节点（最优节点因线路而异，可自定义 host）。
